Adding new DOM elements

Steps to add a new DOM element.

  • Create the element using .createElement()
  • Create text that goes inside the elmenet using .createTextNode()
  • Add text to the new element using .appendChild()
  • Add the new element to an existing element using .appendChild()

Adding a new DOM element

This creats a new element titled <figcaption> with text inside

If you check dev tools, you'll see the new code has been added to the DOM.

<figure class="featured-image">
<img src="../_img/note.jpg">
</figure>


//Create a variable for the existing element
//You'll need this to append the new element
const FEATURED = document.querySelector(".featured-image");

//Create a figcaption element
//This creates a fig caption document that is not placed anywhere in the document.
//It's just sitting in the memory of the browser.
var captionElement = document.createElement("figcaption");

//Create text to add to the new figcaption element
var captionText = document.createTextNode("new text to add");

//Add the new figcaption text to the new figcaption element
captionElement.appendChild(captionText);

//Add new figcaption element to the featured image figure
FEATURED.appendChild(captionElement);

Adding to the DOM with a template literal

This adds HTML after a div tag with the class of movements.

Assign a variable to the .movements element.

Next is an Object with a nested Array namved 'movements'.

Create a Function expression that contains a forEach loop

This loop contains variables that are used in a template literal.

The template literal builds the new HTML.

Call the original variable for the DOM element. Use insertAdjacentHTML to add HTML just after the beginning of this element.

This takes 2 arguments. The 1st is position to attach the HTML (afterbegin).

The 2nd is the HTML string you are inserting.

<div class="movements"></div>
  
const containerMovements = document.querySelector('.movements');

const account1 = {
  movements: [200, 450, -400]
};

const displayMovements = function (x) {
  x.forEach(function (mov, i) {

    const type = mov > 0 ? 'deposit' : 'withdrawl'

    const html = `<div class="movements__row">
    <div class="movements__type movements__type--${type}">${i + 1} ${type}</div>     
    <div class="movements__value">${mov}</div>
    </div>
    `;
    // add this to the element you're adding to
    // 2 arguments
    // 1st is position to attach the HTML
    // 2nd is string you are inserting
    containerMovements.insertAdjacentHTML('afterbegin', html)
  });
}

displayMovements(account1.movements)
