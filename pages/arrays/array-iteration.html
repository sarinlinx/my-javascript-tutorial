forEach()

Print only the values

The forEach() method calls a function (a callback function) once for each array element.

The function takes 3 arguments:

  • The item value
  • The item index
  • The array itself

syntax: arrayName.forEach(functionName)

This example uses only the value parameter since that is what is specifically typed out.

Why does txt need to be before 'value' when outputting?

var txt = "";
var numbers = [45, 4, 9, 16, 25];

//call the function myFunction() once for each element using forEach()
//syntax: ARRAY.forEach(functionName)
//This call MUST BE BEFORE the function or nothing is called
numbers.forEach(myFunction);

document.getElementById("demo").innerHTML = txt;

function myFunction(value, index, array) {
  //This only prints the 'value' since that's what's specifically typed out
  //txt is necessary before the value, otherwise only the last element would be displayed
  txt = txt + value + "<br>";
}
Output

This example outputs: 45, 4, 9, 16, 25

map()

Perform a function on each array element

map() method creates a new array by performing a function on each array element.

The map() method does NOT change the original array.

The array must have values for this to work.

This example multiplies each array value by 2, creates a new array, then prints that new array.

var numbers1 = [45, 4, 9, 16, 25];
//Create new array using map()
//Run the function to create the new array
var numbers2 = numbers1.map(myFunction);

document.getElementById("demo").innerHTML = numbers2;

function myFunction(value, index, array) {
    return value * 2;
}
Output

This example outputs:

90,8,18,32,50

The index and array params

Just as above, the index and array parameters can be added or omitted.

filter()

filter() creates a new array, but only with elements that pass a specific test in the function.

The original array is not touched.

This example creates a new array from elements with a value larger than 18

var numbers = [45, 4, 9, 16, 25];
var over18 = numbers.filter(myFunction);

function myFunction(value) {
  return value > 18;
}

console.log(over18);
Output

This example outputs:

Array [ 45, 25 ]

The index and array params

Just as above, the index and array parameters can be added or omitted.

Searching an Array Object for a string

This searches the Array named 'notes' for a string.

If found, the Object within that string is returned (from the new Array created by filter())

//Create an Array of Objects
const notes = [{
    title: 'My next trip',
    body: 'I would like to go to Spain'
}, {
    title: 'Habbits to work on',
    body: 'Exercise. Eating a bit better.'
}, {
    title: 'Office modification',
    body: 'Get a new seat'
}]


//filter notes Array based on a string
//notesArray is the Array param you're sending in
//query is the string you're searching for
//findNotes will contain the new Array created by filter()
const findNotes = function (notesArray, query) {
    //return a new array based off a new function
    //1 params which is the Array Object
    //This will return a boolean
    //true means it will be kept and sent to the new array
    return notesArray.filter(function (notesObj) {
        //look at the title of the Object being passed into the function
        //Method chaining - chain a 2nd method to the first method in one line
        //includes() checks if the title includes a string noted by 'query'
        const isTitleMatch = notesObj.title.toLowerCase().includes(query.toLowerCase())
        //Do the same for body
        const isBodyMatch = notesObj.body.toLowerCase().includes(query.toLowerCase())
        //return true if title or body matches
        return isTitleMatch || isBodyMatch
    })
}

//Send the Array 'notes' into the function 'findNotes'
//Send the string to search for into the function also
console.log(findNotes(notes, 'eating'))
Output

This example outputs:

Object { title: "Habbits to work on", body: "Exercise. Eating a bit better." }

every()

It returns true or false.

console.log(allOver18);

indexOf()

indexOf() searches an array for an element value and returns its position.

Syntax: array.indexOf(item, start)

The 'item' is required.

'start' is optional. It's where to start searching from. Negative values will start at the given position counting from the end, and search to the end.

indexOf() returns -1 if the item isn't found.

If the item is present more than once, it returns the position of the first occurrence.

var fruits = ["Apple", "Orange", "Apple", "Mango"];
var a = fruits.indexOf("Apple");
//You don't need to +1, but it helps to show the actual position (not counting from zero)
console.log(a + 1);
Output

This example outputs:

1

lastIndexOf()

lastIndexOf() does the same as indexOf(), but searches from the end.

var fruits = ["Apple", "Orange", "Apple", "Mango"];
var a = fruits.lastIndexOf("Apple");
//You don't need to +1, but it helps to show the actual position (not counting from zero)
console.log(a + 1);
Output

This example outputs:

3

find()

find() returns the value of the first array element that passes a test function.

This example finds (returns the value of ) the first element that is larger than 18.

var numbers = [4, 9, 16, 25, 29];
var first = numbers.find(myFunction);

function myFunction(value, index, array) {
    return value > 18;
}

console.log(first);
Output

This example outputs:

25
