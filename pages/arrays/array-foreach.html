forEach()

Print only the values

The forEach() method calls a function (a callback function) once for each array element.

A callback function is that gets passed into another function.

The function takes 3 arguments:

  • The item value
  • The item index
  • The array itself

syntax: arrayName.forEach(functionName)

This example uses only the value parameter since that is what is specifically typed out.

Notice the callback function is called within the forEach() API. You do not call the callback function directly.

Why does txt need to be before 'value' when outputting?

var txt = "";
var numbers = [45, 4, 9, 16, 25];

//call the function myFunction() once for each element using forEach()
//syntax: ARRAY.forEach(functionName)
//This call MUST BE BEFORE the function or nothing is called
numbers.forEach(myFunction);

document.getElementById("demo").innerHTML = txt;

function myFunction(value, index, array) {
  //This only prints the 'value' since that's what's specifically typed out
  //txt is necessary before the value, otherwise only the last element would be displayed
  txt = txt + value + "<br>";
}
Output

This example outputs: 45, 4, 9, 16, 25

Since only the 'value' is specifically printed, you could rewrite the function to remove 'index' and 'array'.

var txt = "";
var numbers = [45, 4, 9, 16, 25];

//call the function myFunction() once for each element using forEach()
//syntax: ARRAY.forEach(functionName)
//This call MUST BE BEFORE the function or nothing is called
numbers.forEach(myFunction);

document.getElementById("demo").innerHTML = txt;

function myFunction(value) {
  //This only prints the 'value' since that's what's specifically typed out
  //txt is necessary before the value, otherwise only the last element would be displayed
txt = txt + value + "<br>";
}

Print the values and index

This does the same as above, but it also prints out the index#.

You just need to specify it within the function output.

var txt = "";
var numbers = [45, 4, 9, 16, 25];

numbers.forEach(myFunction);

document.getElementById("demo").innerHTML = txt;

function myFunction(value, index, array) {
  txt = txt + "Value is: " + value + " Index is: " + index + "<br>";
}
Output

This example outputs:

Value is: 45 Index is: 0
Value is: 4 Index is: 1
Value is: 9 Index is: 2
Value is: 16 Index is: 3
Value is: 25 Index is: 4

Print the values, index, and array

This does the same as above, but it also prints out the full array.

var txt = "";
var numbers = [45, 4, 9, 16, 25];

numbers.forEach(myFunction);

document.getElementById("demo").innerHTML = txt;

function myFunction(value, index, array) {
txt = txt + "Value is: " + value + " Index is: " + index + " Array is: " + array + "<br>";
}
Output

This example outputs:
