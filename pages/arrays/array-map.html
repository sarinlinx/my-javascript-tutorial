map()

Perform a function on each array element

map() method creates a new array by performing a function on each array element.

The map() method does NOT change the original array.

The array must have values for this to work.

This example multiplies each array value by 2, creates a new array, then prints that new array.

var numbers1 = [45, 4, 9, 16, 25];
//Create new array using map()
//Run the function to create the new array
var numbers2 = numbers1.map(myFunction);

document.getElementById("demo").innerHTML = numbers2;

function myFunction(value, index, array) {
    return value * 2;
}
Output

This example outputs:

90,8,18,32,50

The index and array params

Just as above, the index and array parameters can be added or omitted.
