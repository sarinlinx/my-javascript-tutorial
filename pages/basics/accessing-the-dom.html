The BOM and DOM

All web pages are organized by the BOM and DOM.

The BOM

BOM is the 'Browser Object Model'. The 'window' is the top level object within the BOM.

View the following link for properties and methods to use with the 'window' object.

  • Properties and Methods of the BOM

The DOM

The DOM is inside the BOM.

The DOM is a model for HTML that forms the web page.

The document is one giant object. And all the elements within the document are nodes inside that object.

View the following link for properties and methods of the Document.

  • Properties and Methods of the Document

For example, if you wanted to access the title node, you'd use: document.title.

Methods to retrieve elements

Older methods

These are old methods which are good for forms or if you want your code to be unambiguous.

Get elements with specific ID.

document.getElementById("some-ID");

Get all element with a specified class name as array.

document.getElementsByClassName("classname");

Get all elements with specified HTML tag as array.

document.getElementsByTagName("HTML tag");

Newer methods

These are easier to use.

querySelector()

These must be triggered. A button click is used below.

querySelector. Get the first element matching specified selectors.

  • querySelector
document.querySelector(".main-nav a");

Retrieve the text inside the element (by class)

This example retrieves the HTML inside the element.

This returns: <p>Main Title</p>

Note that everything inside the .main-title class is returned, even the paragraph tags.

<div class="main-title"><p>Main Title</p></div>
<button onclick="myFunction()">Click to change</button>

function myFunction() {
  var innerhtml = document.querySelector(".main-title").innerHTML;
  alert(innerhtml);
}

Retrieve the HTML and text of an element (by class)

This example retrieves the HTML around the element.

This returns: <div class="main-title"><p>Main Title</p></div>

<div class="main-title"><p>Main Title</p></div>
<button onclick="myFunction()">Click to change</button>

function myFunction() {
  var outerhtml = document.querySelector(".main-title").outerHTML;
  alert(outerhtml);
}

Change the text inside an element (by class)

This example retrieves the HTML inside the element, then uses innerHTML = to change it to something else.

This returns: <p>New Title</p>

<div class="main-title">Main Title</div>
<button onclick="myFunction()">Click to change</button>

function myFunction() {
  var innerhtml = document.querySelector(".main-title").innerHTML = "New Title";
}

Change the ID of an element

This changes the 'id' attribute of an element.

This changes the id of 'firstID' to 'secondID'.

You can see this since the second alert box is different.

<p id="firstID">First paragraph</p>
<button onclick="myFunction()">Click to change</button>

function myFunction() {
  var firstID = document.querySelector("#firstID").id;
  alert(firstID);
  var firstID = document.querySelector("#firstID").id = "secondID";
  alert(firstID);
}

Interacting with a read-only properties

This page lists all read-only properties.

  • Read only properties

Return a list of all classes of an element

Use .classList to return a list of all classes that are assigned to an element.

classList is read-only, so these values can't be changed this way.

This example returns: masthead clear.

<header class="masthead clear">header here</header>
<button onclick="myFunction()">Click to change</button>

function myFunction() {
var classList = document.querySelector(".masthead").classList;
alert(classList);
}

You could also return a specific class by using the index value.

This returns 'clear'.

<header class="masthead clear">header here</header>
<button onclick="myFunction()">Click to change</button>

function myFunction() {
var classList = document.querySelector(".masthead").classList[1];
alert(classList);
}

How to change the class of a read-only property like classList?

You must use a method to change the class of a read-only property list classList.

Add a class name

Use .add("classname") to add another class to an element.

This returns: masthead clear newClassName

<header class="masthead clear">header here</header>
<button onclick="myFunction()">Click to change</button>

function myFunction() {
  document.querySelector(".masthead").classList.add("newClassName");
  var classList = document.querySelector(".masthead").classList;
  alert(classList);
}

Remove a class

Use .remove("classname") to remove a class from an element.

This returns: masthead since the class of 'clear' was removed.

<header class="masthead clear">header here</header>
<button onclick="myFunction()">Click to change</button>

function myFunction() {
  document.querySelector(".masthead").classList.remove("clear");
  var classList = document.querySelector(".masthead").classList;
  alert(classList);
}

Toggle a class on/off

Use .toggle("classname") to toggle on/off a class from an element.

Use querySelector() to locate a different class from the element, then toggle() to toggle another.

This toggles off the 'clear' class.

<header class="masthead clear">header here</header>
<button onclick="myFunction()">Click to change</button>

function myFunction() {
document.querySelector(".masthead").classList.toggle("clear");
  var classList = document.querySelector(".masthead").classList;
  alert(classList);
}

Check if another class if part of an element

Use .contains("classname") to check if an existing element with a class, also contains a different class.

In this example, the header element contains a class of 'masthead'. .contains() then checks if this element aslo contains the class of 'clear'.

If so, it returns TRUE.

<header class="masthead clear">header here</header>
<button onclick="myFunction()">Click to change</button>

function myFunction() {
document.querySelector(".masthead").classList.contains("clear");
  var classList = document.querySelector(".masthead").classList;
  alert(classList);
}

Changing attributes

Syntax is: var result = elementName.hasAttribute(name);

'result' holds either TRUE or FALSE.

'name' is a string of the name of the attribute.

Check if an element has a specific attribute (hasAttribute)

Checking by the ID

This checks if an element with an id of 'headerID' also contains a 'class' attribute.

This returns TRUE.

View the following link for further details.

  • hasAttribute
<header id="headerID" class="masthead">header here</header>
<button onclick="myFunction()">Click to change</button>

function myFunction() {
  //first get the element
  var hasAttr = document.getElementById("headerID").hasAttribute("class");
  alert(hasAttr);
}

Checking using querySelector

This checks if the 'header' element contains a 'class' attribute.

This returns TRUE.

View the following link for further details.

<header class="masthead">header here</header>
<button onclick="myFunction()">Click to change</button>

function myFunction() {
  //first get the element
  var hasAttr = document.querySelector("header").hasAttribute("class");
  alert(hasAttr);
}

getAttribute()

Return the attribute names of an element as an Array of strings

This first looks for the element with an id of 'paragraphID'.

It then checks if the attribute 'class' also exists for this same element.

If so, it returns the string name of that attribute.

This example returns "myClass".

View the following for more info:

  • getAttribute
<p id="paragraphID" class="myClass"><a href="#">header here</a></p>
<button onclick="myFunction()">Click to change</button>

function myFunction() {
  var hasAttr = document.getElementById("paragraphID").getAttribute("class");
  alert(hasAttr);
}

Another example.

This returns "_blank".

<a id="myAnchor" href="#" target="_blank">Attr object</a>
<button onclick="myFunction()">Click to change</button>

function myFunction() {
  var hasAttr = document.getElementById("myAnchor").getAttribute("target");
  alert(hasAttr);
}

setAttribute()

Sets the value of an attribute on the specified element

Sets the value of an attribute on the specified element. If the attribute already exists, the value is updated; otherwise a new attribute is added with the specified name and value.

The retrieves the element with a class of 'paragraphID', then adds a new class of 'mynewClass'.

  • setAttribute
<p id="paragraphID" class="myClass"><a href="#">header here</a></p>
<button onclick="myFunction()">Click to change</button>

function myFunction() {
  document.getElementById("paragraphID").setAttribute("class", "mynewClass");
  var getNewClass = document.querySelector("#paragraphID").classList;
  alert(getNewClass);
}

This retrieves the element with an id of 'myAnchor'.

It then adds a new attribute of 'href' with a value of 'example.com'.

<a id="myAnchor">A Link: Go to w3schools.com</a>
<button onclick="myFunction()">Click to change</button>

function myFunction() {
  document.getElementById("myAnchor").setAttribute("href", "https://www.example.com");
}
