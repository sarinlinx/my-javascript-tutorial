Default TRUE (truthy) values

The following values are always truthy:

  • '0' (a string containing a single zero)
  • 'false' (a string containing the text “false”)
  • [] (an empty array)
  • {} (an empty object)
  • function(){} (an “empty” function)

Default FALSE (falsy) values

The following values are always falsy:

  • false
  • 0 (zero)
  • '' or "" (empty string)
  • null
  • undefined
  • NaN (e.g. the result of 1/0)

AND

AND returns last TRUE value in a series of TRUE

// Left operand is true,
// AND needs to check the right operand also
// It then returns that last TRUE value

alert(1 && 0) // 0, the last evaluated value
alert(1 && 5) // 5, the last evaluated value

AND returns FALSE if the first value is FALSE.

// Left operand is false,
// AND stops immediately and returns it

alert(null && 5) // Since the first value if FALSE (null), it is the last evaluated value

Logical OR with falsy operands

When using OR with falsy operands, it looks for the TRUE value.

In this example, the left operand is TRUE, so OR stops and returns it immediately.

The result is 1.

alert(1 || 0)

In this example, the left operand is FALSE, so OR continues to the next operand which is TRUE and returns it.

The result is 1.

alert(0 || 1)

If both are false

In this example, the left operand is FALSE, so OR continues to the next operand which is also FALSE.

Since there is no TRUE value, the result is null.

alert(0 || 1)

When to use?

This is useful when you have multiple variables and need to get the first one that is true.

This returns 'Oh-la-la' since it's the last value that isn't falsy.

var b = false
var n = 0
var s = "O-la-la"

var result = b || n || s

alert(result) // "O-la-la"

NOT

NOT converts its operand into boolean, then negates it.

alert( !true ) // returns false
alert( !0 ) // returns true

Double NOT

A Double NOT converts its value into boolean.

alert( !!"string" ) //	returns true
alert( !!null ) // returns false

Boolean conversion

Traditionally, it is possible to put a value into if. In this case, it gets converted into boolean.

var x = 1

if (x) {
  alert('true')
}
