Strings

Strings are text.

In JavaScript, all strings are in unicode, no matter which type of encoding is used.

Basics

Creating a string

Put the string in quotes.

var text = "my text string";

Adding a number and a string

The result is 45 because the number is just joined with the string.

If you check typeof, you'll see it's a string now.

var a = 4;
var b= "5";
var sum = a + b;

alert(sum);
alert(typeof sum);

Escaping quotes

To escape a quote, add a \ slash before the quote.

Escape a single quote in a single quoted string

A single quote inside a single quoted string must be escaped.

var str = 'I\'m the Walrus';
alert(str)

Escape a single quote in a double quoted string

A single quote inside a double quoted string does not need to be escaped.

var str = "I'm the Walrus";
alert(str)

Escape double quotes in a double quoted string

A double quote inside a double quoted string must be escaped.

var str = "double \"quote\" ";
alert(str)

String length

Print the number of characters in a string.

This prints 3 since the newline is its own character.

var str = "My\n"
alert(str.length)

Accessing a single character in a string

There are 2 ways to do this.

var str = "string text"
alert( str.charAt(2) )

var str = "string text"
alert(str[2])

Changing string text

It's not possible to change the text of a string.

However, you can re-created the string one part at a time.

This changes the text 'string' into 'ring'.

var str = "string"
str = str.charAt(2) + str.charAt(3) + str.charAt(4) + str.charAt(5)
alert(str)
