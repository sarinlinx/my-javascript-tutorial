Converting a String to an Array

comma

Display the first array element, after a string split from a comma.

This returns: a

var str = "a,b,c,d,e,f";
var arr = str.split(",");

pipe

Display the first array element, after a string split from a |.

This returns: a,b,c

var str = "a,b,c|d,e,f";
var arr = str.split("|");

space

Display the first array elements, after a string split from a space.

This returns: a,b,c,d,e

var str = "a,b,c,d,e f";
var arr = str.split(" ");
