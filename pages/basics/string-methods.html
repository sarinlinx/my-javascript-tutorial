String methods

With JavaScript, methods and properties are available to primitive values, because JavaScript treats primitive values as objects when executing methods and properties.

All string methods return a new string. They don't modify the original string. Formally said: Strings are immutable: Strings cannot be changed, only replaced.

String length

Print the number of characters in a string.

This prints 3 since the newline is its own character.

var str = "My\n"
  alert(str.length)

Searching for the index value String in a String

indexOf()

Use indexOf() to search for a sub-string within a string.

It returns the position of a first occurence of a substring or -1 if nothing found.

This example returns the following:

  • returns 0
  • returns 1 because the second character starts with 'id'
  • returns -1
var str = "Widget with id"

    alert( str.indexOf("Widget") )
    alert( str.indexOf("id") )
    alert( str.indexOf("Lalala") )

Find sub-string after beginning at a specific index

This starts searching at character 2.

It returns 12 because that's the first character position (after 2) that starts with 'id'.

var str = "Widget with id"
    alert( str.indexOf("id", 2) )

Finding the first index value of a String in a String (indexOf)

The indexOf() method returns the index of (the position of) the first occurrence of a specified text in a string.

This returns: 7

var str = "Please locate where 'locate' occurs!";
            var pos = str.indexOf("locate");

Finding the last index value of a String in a String (lastIndexOf)

The lastIndexOf() method returns the index of the last occurrence of a specified text in a string.

This returns 21 since it finds the second (or last) occurrence of the word 'locate'.

var str = "Please locate where 'locate' occurs!";
            var pos = str.lastIndexOf("locate");

If a String in a String doesn't exist

Both indexOf(), and lastIndexOf() return -1 if the text is not found.

var str = "Please locate where 'locate' occurs!";
            var pos = str.lastIndexOf("John");

Setting a starting index

Both methods accept a second parameter as the starting position for the search.

Using search() instead of indexOf()

The search() method searches a string for a specified value and returns the position of the match.

This returns 7

Difference between search() and indexOf()

The search() method cannot take a second start position argument.

The indexOf() method cannot take powerful search values (regular expressions).

Extracting a Sub-string from a String

substring()

substring() extracts a sub-string from a string using index values.

substring() cannot use negative values.

The first value is the index it begins to cut from.

The last index value is what index to stop at, but does not include that index value.

Find a sub-string between specific characters

This prints the letter 's'.

It starts at index 0, then cuts out 1 index value.

var str = "stringify"
alert(str.substring(0,1))

Another example of two values

It starts at index 2, stops cutting at 4 (which is not included. So only index 2 and 3 are cut.

This prints the letters 'ri'.

var str = "stringify"
alert(str.substring(2,4))

What if only 1 index value is specified?

If only 1 index value is given, it starts cutting from there to the end of the string.

This prints ringify since only the first value was specified.

var str = "stringify"
alert(str.substring(2))

What if first value is larger than second?

If the first value is larger, the values are swapped.

-1 becomes zero, then the values are swapped.

This returns 'stri'.

var str = "stringify"
alert(str.substring(4, -1))

substr()

substr() is not the same as substring()

With substr(), the second parameter specifies the length of the extracted part.

Syntax

substr(beginning_value , how_many_to_extract)

Find a sub-string starting at a specific character

This starts at index 2, then cuts 4 index values.

This prints 'ring'.

var str = "stringify"
str = str.substr(2,4)
alert(str)

What if only 1 index value is given?

It will start at the index value and cut everything after.

This prints: Banana, Kiwi

var str = "Apple, Banana, Kiwi";
var res = str.substr(7);

If index value is negative

It counts backwards from the end of the string.

This prints: gify

var str = "stringify";
str = str.substr(-4);

If a negative and positive index is used

It counts backwards from the end of the string.

The 2nd index counts how many to cut.

This prints: gi

var str = "stringify";
str = str.substr(-4, 2);

Two negative values will not work with substr()

slice()

slice() extracts a part of a string and returns the extracted part in a new string.

You can use a single value, or 2 values.

The values and be positive or negative.

The first value is where it starts to cut from.

The second index value is where to stop (but does not include that index value).

Using a single value with slice()

The index is where it begins to cut from.

This returns: Banana, Kiwi

var str = "Apple, Banana, Kiwi ";
var res = str.slice(7);

Using two index values with slice()

The method takes 2 parameters: the starting index (position), and the ending index (position).

The first index is where it begins to cut from.

The last index is where it stops, meaning it does not cut that final value.

This example slices out a portion of a string from position 7 to position 13:

This returns 'Banana'.

var str = "Apple, Banana, Kiwi ";
var res = str.slice(7, 13);

Using a single negative value with slice()

When couting, do not use 0 as the first value. Count back starting at 1.

If a single negative value is specified, you count backwards to that index, then cut the rest of the string out.

This returns: Kiwi

var str = "Apple, Banana, Kiwi";
var res = str.slice(-4);

Using two negative values with slice()

If a parameter is negative, the position is counted from the end of the string.

This example slices out a portion of a string from position -12 to position -6:

When couting, do not use 0 as the first value. Count back starting at 1.

The first index is where the string starts to cut from.

The last index is where it stops, so it is not included in the returned string.

This returns: Banana

var str = "Apple, Banana, Kiwi";
var res = str.slice(-12, -6);

A negative with a positive value will not work with slice()

Difference between slice() and substring()

The difference between slice() and substring() is how they treat negative and overflow values.

Negative values using substring() become zero.

Negative values using slice() count backwards.

This returns 'testme' because -2 becomes 0, and there's no second value.

alert( "testme ".substring(-2) )

slice is the same with positive values.

This returns 'test'.

alert ( "testme ".slice(0, 4));

With slice() and negative values, it counts back from the end.

This returns 'me'.

alert( "testme ".slice(-2) )

why does this return 'm'?

alert ( "testme ".slice(4, -1));

Replacing content with replace()

The replace() method replaces a specific sub-string with another string.

By default, the replace() function replaces only the first match.

By default, the replace() function is case sensitive.

The replace() method does not change the string it is called on. It returns a new string.

This replaces 'cake' with 'pie'.

This returns: I like pie!

str = "I like cake!";
var n = str.replace("cake", "pie");

Replace case-sensitive text

Use a regular expression with an /i flag (insensitive)

Do not use quotes.

str = "I like CAKE!";
var n = str.replace(/cAke/i, "pie");

Replace all instances of a word

To replace all matches, use a regular expression with a /g flag (global match).

The /g flag is case sensitive. Add /i to find all references.

str = "I like CAKE with extra cake!";
var n = str.replace(/cake/gi, "pie");

Altering Strings

charAt()

It's not possible to change the text of a string.

However, you can re-created the string one part at a time.

This changes the text 'string' into 'ring'.

var str = "string"
  str = str.charAt(2) + str.charAt(3) + str.charAt(4) + str.charAt(5)
  alert(str)

Changing string case

Uppercase

Use the method toUpperCase to change the string to uppercase text.

alert( "Hey-ho!".toUpperCase() )

Lowercase

Use the method toLowerCase to change the string to lowercase text.

alert( "Hey-ho!".toLowerCase() )

Concat

concat() joins two or more strings.

var text1 = "Hello";
var text2 = "World!";
var text3 = text1.concat(" ",text2);

The concat() method can be used instead of the plus operator. These two lines do the same.

var text = "Hello" + " " + "World!";
var text = "Hello".concat(" ", "World!");

trim()

String.trim() removes whitespace from both sides of a string.

var str = "    Hello World!    ";
alert(str);

includes()

This returns 'false' since 'xyz' is not part of the string named 'password'.

let password = 'abc123asdf098'
console.log(password.includes('xyz'))

Example of includes() and length()

This returns 'false' since 'xyz' is not part of the string named 'password'.

This returns: false, true, false.

let isValidPassword = function (password) {
  //length must be over 8 char AND cannot contain the string xyz
  return password.length > 8 && !password.includes('xyz')
}

//Returns true because it's under 8 char
console.log(isValidPassword('asdfp'))
//Returns true because it's over 8 char, and it DOES NOT contain 'xyz'
console.log(isValidPassword('abc123!@#$%^z'))
//Returns false because DOES contain the string 'xyz'
console.log(isValidPassword('asdfpasdfpoij_xyz'))

ECMAScript 5 (2009) Property Access

ECMAScript 5 (2009) allows property access [ ] on strings.

Property access might be a little unpredictable:

  • It makes strings look like arrays (but they are not)
  • If no character is found, [ ] returns undefined, while charAt() returns an empty string.
  • It is read only. str[0] = "A" gives no error (but does not work!)

This returns: H

var str = "HELLO WORLD";
str[0];
