String methods

With JavaScript, methods and properties are available to primitive values, because JavaScript treats primitive values as objects when executing methods and properties.

String length

Print the number of characters in a string.

This prints 3 since the newline is its own character.

var str = "My\n"
  alert(str.length)

Searching for the index value String in a String

indexOf()

Use indexOf() to search for a sub-string within a string.

It returns the position of a first occurence of a substring or -1 if nothing found.

This example returns the following:

  • returns 0
  • returns 1 because the second character starts with 'id'
  • returns -1
var str = "Widget with id"

    alert( str.indexOf("Widget") )
    alert( str.indexOf("id") )
    alert( str.indexOf("Lalala") )

Find sub-string after beginning at a specific index

This starts searching at character 2.

It returns 12 because that's the first character position (after 2) that starts with 'id'.

var str = "Widget with id"
    alert( str.indexOf("id", 2) )

Finding the first index value of a String in a String (indexOf)

The indexOf() method returns the index of (the position of) the first occurrence of a specified text in a string.

This returns: 7

var str = "Please locate where 'locate' occurs!";
            var pos = str.indexOf("locate");

Finding the last index value of a String in a String (lastIndexOf)

The lastIndexOf() method returns the index of the last occurrence of a specified text in a string.

This returns 21 since it finds the second (or last) occurrence of the word 'locate'.

var str = "Please locate where 'locate' occurs!";
            var pos = str.lastIndexOf("locate");

If a String in a String doesn't exist

Both indexOf(), and lastIndexOf() return -1 if the text is not found.

var str = "Please locate where 'locate' occurs!";
            var pos = str.lastIndexOf("John");

Using search() instead of indexOf()

The search() method searches a string for a specified value and returns the position of the match.

This returns 7

var str = "Please locate where 'locate' occurs!";
            var pos = str.search("locate");

Difference between search() and indexOf()

The search() method cannot take a second start position argument.

The indexOf() method cannot take powerful search values (regular expressions).

substring()

Find a sub-string starting at a specific character

Syntax

substr(beginning_value , how_many_to_extract)

This prints 'ring'.

var str = "stringify"
    str = str.substr(2,4)
    alert(str)

Find a sub-string between specific characters

Syntax

substring(beginning_value , end_value)

This prints the letter 's'.

var str = "stringify"
    alert(str.substring(0,1))

This prints ringify since only the first value was specified.

var str = "stringify"
    alert(str.substring(2))

What if first value is larger than second?

If the first value is larger, the values are swapped.

-1 becomes zero, then the values are swapped.

This returns 'test'.

alert( "testme".substring(4, -1) )

slice()

slice() extracts a part of a string and returns the extracted part in a new string.

You can use a single value, or 2 values.

The values and be positive or negative.

Using a single value with slice()

The index is where it begins to cut from.

This returns: Banana, Kiwi

var str = "Apple, Banana, Kiwi";
    var res = str.slice(7);

Using two index values with slice()

The method takes 2 parameters: the starting index (position), and the ending index (position).

The first index is where it begins to cut from.

The last index is where it stops, meaning it does not cut that final value.

This example slices out a portion of a string from position 7 to position 13:

This returns 'banana'.

var str = "Apple, Banana, Kiwi";
    var res = str.slice(7, 13);

Using negative values with slice()

Using a single negative value with slice()

When couting, do not use 0 as the first value. Count back starting at 1.

If a single negative value is specified, you count backwards to that index, then cut the rest of the string out.

This returns: Kiwi

var str = "Apple, Banana, Kiwi";
    var res = str.slice(-4);

Using two negative values with slice()

If a parameter is negative, the position is counted from the end of the string.

This example slices out a portion of a string from position -12 to position -6:

When couting, do not use 0 as the first value. Count back starting at 1.

The first index is where the string starts to cut from.

The last index is where it stops, so it is not included in the returned string.

This returns: Banana

var str = "Apple, Banana, Kiwi";
    var res = str.slice(-12, -6);

Difference between slice() and substring()

The difference between slice() and substring() is how they treat negative and overflow values.

Negative values using substring() become zero.

Negative values using slice() count backwards.

This returns 'testme' because -2 becomes 0, and there's no second value.

alert( "testme".substring(-2) )

slice is the same with positive values.

This returns 'test'.

alert ( "testme".slice(0, 4));

With slice() and negative values, it counts back from the end.

This returns 'me'.

alert( "testme".slice(-2) )
