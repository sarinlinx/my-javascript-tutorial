Creating Objects

Objects store data such as properties and methods.

  • Properties: Info about the object we can retrieve and use
  • Methods: functions that belong to the object

Object conversion

Avoid String, Number, and Boolean objects. They complicate your code and slow down execution speed.

var x = new String(); // Declares x as a String object
var y = new Number(); // Declares y as a Number object
var z = new Boolean(); // Declares z as a Boolean object

Objects in JavaScript can be converted to primitives in three contexts:

  • Numeric
  • String
  • Boolean

Comparing Objects to each other using ===

Comparing two objects with === does not work. This is because even if the Objects have the exact same properties and values, they are stored in memory in different locations.

What makes them equal is if they are the same object stored in memory.

For example, this will never work.

console.log( {} === {})
