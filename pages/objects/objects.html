Creating Objects

Objects store data such as properties and methods.

  • Properties: Info about the object we can retrieve and use
  • Methods: functions that belong to the object

Long way to create an Object

Use {} to create an Object.

When a JavaScript variable is declared with the keyword "new", the variable is created as an object. However this isn't necessary since you can assign properties while creating it.

This creates and Object named 'course'.

var course = new Object();

You can then run commands to assign it properties using dot or bracket notation.

course.title = "JavaScript Essential Training";
course.instructor = "Name of teacher";
course.level = 1;
course["Where it is"] = "At the school";

Short way to create an Object

The short hand way is to assign the properties while creating the Object.

var course = {
  title: "JavaScript Essential Training",
  instructor: "Name of teacher",
  level: 1,
  views: 0,
}

Retrieving data from Objects

Retrieving the full Object

You can retrieve the full Object just like a variable.

console.log(course);

Retrieving properties using dot notation

You can use dot notation to retrieve a specific property.

The syntax is: objectName.propertyName

Dot notation works well for properly named properties.

console.log(course.title);

Retrieving properties using bracket notation

You can use bracket notation to retrieve a specific property.

The syntax is: objectName["propertyName"]

Bracket notation works well for when we might have to convert the property name into a string.

console.log(person["lastName"];);

Deleting an Object or property

Deleting a property

You can delete a property by running 'delete' following by the property value in dot notation.

delete course.title;

Empty an Object

You can empty an object by setting it to null.

If you now run 'person.firstName' it will return null.

var person = { firstName:"John",
  lastName:"Doe", age:50,
  eyeColor:"blue"
};

var person = null;

Object conversion

Avoid String, Number, and Boolean objects. They complicate your code and slow down execution speed.

var x = new String(); // Declares x as a String object
var y = new Number(); // Declares y as a Number object
var z = new Boolean(); // Declares z as a Boolean object

Objects in JavaScript can be converted to primitives in three contexts:

  • Numeric
  • String
  • Boolean

Iterating over Object key:value pairs

You can loop through key:value pairs by using a for loop with the 'key' keyword.

var menu = {
  width:  300,
  height: 200,
  title: "Menu"
};

for(var key in menu) {
var val = menu[key];

console.log("Key: " + key + " \nValue: " + val);
}

The code above prints out the following:

Key: width
Value: 300

Key: height
Value: 200

Key: title
Value: Menu
