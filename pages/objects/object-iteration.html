Iterating over Object key:value pairs

You can loop through key:value pairs by using a for loop with the 'key' keyword.

var menu = {
  width:  300,
  height: 200,
  title: "Menu"
};

for(var key in menu) {
var val = menu[key];

console.log("Key: " + key + " \nValue: " + val);
}

The code above prints out the following:

Key: width
Value: 300

Key: height
Value: 200

Key: title
Value: Menu
