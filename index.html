Arrays

    Basics
    • Arrays - Overview and length property
    Array Iteration
    • Array for loop
    • forEach() - calls a function (a callback function) once for each array element
    • indexOf() - returns index of an element
    • map() - creates a new array by performing a function on each array element
    • filter() - creates a new array, but only with elements that pass a specific test in the function
    • every() - check if all array values pass a test
    • some() - check if any array values pass a test
    • lastIndexOf() - does the same as indexOf(), but searches from the end.
    • find() - returns the value of the first array element that passes a test function
    • findIndex() - returns the index of the first array element that passes a test function
