HTML

Clicking the dropdown displays the value assigned to it in the console.

<select id="filter-by">
  <option value="byEdited">Sort by last edited</option>
  <option value="byCreated">Sort by recently created</option>
  <option value="alphabetical">Sort alphabetically</option>
</select>

JS

Open console to view the value of the current dropdown

//target dropdown
//check for its change event
document.querySelector('#filter-by').addEventListener('change', function (e) {
  //this targets the value assigned to the dropdown option in HTML
  console.log(e.target.value)
})
